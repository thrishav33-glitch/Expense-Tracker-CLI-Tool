# Expense Tracker CLI Tool 

A Python command-line application for tracking daily expenses. Users can add expenses, categorize spending, save records to a CSV file, view expense history, and calculate total spending.

## Features
- Add Expenses
- Categorize Expenses
- Save Data to CSV File
- View Expense Records
- Calculate Total Spending
- User-Friendly CLI Interface

## Technologies Used
- Python
- CSV File Handling
